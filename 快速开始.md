## 📝 你需要完成的内容

### 1️⃣ 准备个人照片

- 准备一张你的照片或卡通形象
- 将图片命名为：**photo.jpg**
- 放入 **images/** 目录

### 2️⃣ 填写自我介绍

打开 `index.html` 文件，找到第 48 行左右，在以下位置填写你的自我介绍（不少于 100 字）：

```html
<p class="about-text">
    <!-- 请在此处填写您的自我介绍，不少于 100 字 -->
</p>
```

### 3️⃣ 添加外部链接

打开 `index.html` 文件，找到第 73 行左右，修改链接地址：

```html
<a href="#" class="external-link" target="_blank">
    <!-- 请在此处填写您的外部链接地址 -->
    个人主页
</a>
```

将 `href="#"` 改为你实际的链接，例如：
- GitHub 主页：`https://github.com/yourusername`
- 博客：`https://yourblog.com`
- 其他社交平台链接

### 4️⃣ 推送到 Gitee

#### 步骤 1：在 Gitee 创建仓库

1. 访问 https://gitee.com/
2. 登录账号
3. 点击右上角"+" → "新建仓库"
4. 仓库名称：`my_website_tag`
5. 点击"立即创建"

#### 步骤 2：推送代码

在项目根目录执行以下命令（**将 `<你的用户名>` 替换为你的 Gitee 用户名**）：

```bash
git remote add origin https://gitee.com/<你的用户名>/my_website_tag.git
git push -u origin master
```

#### 步骤 3：启用 Gitee Pages

1. 进入仓库页面
2. 点击"服务" → "Gitee Pages"
3. 选择分支：`master`
4. 点击"保存"

#### 步骤 4：访问你的数字名片

等待几分钟后，访问生成的链接：
```
https://<你的用户名>.gitee.io/my_website_tag/
```
